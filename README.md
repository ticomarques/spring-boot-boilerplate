Este projeto é um boilerplate para iniciar os projetos usando spring boot.

## Dependências
- Starter Web
- JPA
- Validation
- Security
- Oauth2-client
- Oauth2-authorization-server
- Actuator
- Thymeleaf
- WebMVC
- Hibernate

## Instalação

1 - Clone o projeto

2 - Abra pasta na sua IDE

3 - Configure o resources/application.yml e env.vars.

4 - Selecione Java 21 na sua IDE e execute o projeto (Clean e Install).
